⚠️ Modo local (solo este dispositivo) Para que los datos sean visibles desde cualquier dispositivo, configura Firebase Realtime Database. Ver instrucciones →
Cómo conectar Firebase (gratis, 5 min):
  1. Ve a console.firebase.google.com → Crear proyecto → dale un nombre → continuar.
  2. En el proyecto, clic en Realtime Database → Crear base de datos → modo prueba.
  3. Clic en el ícono </> (web) para registrar tu app, copia el objeto firebaseConfig.
  4. Abre el archivo index.html en el editor → busca la variable FIREBASE_CONFIG al inicio del script → pega los valores.
  5. Guarda, sube con git push y listo.
